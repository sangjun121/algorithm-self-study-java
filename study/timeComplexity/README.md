# 시간 복잡도의 모든 것. (JAVA 기준)
- 제한 시간이 1초인 문제는 연산 횟수가 3,000만이 넘는 알고리즘은 사용하면 안된다.

| 시간 복잡도 | N의 가용 범위      |
|--------|---------------|
|   O(N!)     | 10            |
|    O(2ⁿ)    | 20~25         |
| O(N³)       | 200~300       |
|  O(N²)      | 3,000 ~ 5,000 |
| O(NlogN)       | 100만          |
| O(N)       | 1,000만~2,000만 |
| O(logN)       |  10억             |
