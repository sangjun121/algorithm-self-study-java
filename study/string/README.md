# 문자열

### String 주요 메소드

1. ```String.toCharArray()``` -> char 배열로 변환
```java
   //example: 
   char[] chars = input.toCharArray();
 ```
 
2. char -> String으로 변환하기
```java
   Character.toString(char i);
 ```

3. int -> String
```java
   String.valueOf(int i);
 ```
